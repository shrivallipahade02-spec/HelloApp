# HelloApp

Hello App starts with Hello World, progresses to displaying a user 
name, then names from command-line args and standard input. It then 
manages names in a collection with list/remove options, refactors 
into methods and classes, adds persistence across runs, and finally 
displays names in banner format.

## How to Run
1. Compile: `mvn clean install`
2. Run: `java -cp target/hello-app-1.0.jar HelloApp`
